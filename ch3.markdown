# 文件操作

## 排序

在详情模式和加强模式之外的显示模式，用鼠标右键选择「排序方式」或者使用工具栏按钮 可以将文件和文件夹按照名称、大小、类型、修改时间和属性来排序，并且可以选择正序还是反序。

在详情模式和加强模式时，除了以上的方法，和Excel类似，在属性栏点击某一列列名，就可以用该列的数值进行排序，如图，点击「文件总数」之后文件列表就按照文件总数由少到多进行排列，再次点击切换为反序排列。

这里没有使用「升序」还是「降序」，而是用的「正序」还是「反序」，这是因为对每一列属性来说，这两对概念的匹配关系在DOpus里面是可以设置的。比如文件名的默认正序是升序（0-9A-Z），但是修改时间的默认正序是降序（最近的排在最前）。设置的地方在「选项-显示-字段」：

在这个设置窗口里面我们可以定制每个字段的字体以及默认的排序规则。如果默认采取降序的话，字段名称右边会有一个向下的箭头。如果在浏览的时候想直接以反序排列，请按住shift再用鼠标点击列名。

另外，DOpus也支持多重排序，比如我们可以先按照类型排序，然后按住Ctrl键再点击「大小」，就可以先排文件类型，然后在每一个类型里面按照文件大小排序。这个时候参与排序的属性右侧会出现排序的顺位和方向：

## 分组

DOpus可以将文件按照规则分组显示，方便快速定位到文件。从鼠标右键菜单，按钮 以及按住Alt键点击列名都可以打开分组功能。

如果要取消分组的话，在「右键菜单-按…分组」中选择「无」或者按住Shift+Alt点击任何一个列名即可。

## 保存显示设置


经常有朋友在群里面问：「上次我把这个文件夹设置成这个样子，怎么下次我一打开我的设置就没了？」这里我解释一下：你对文件显示栏的操作（添加/删除列、排序、分组）等，如果你不保存的话，其范围仅限于当前标签页。也就是说，只要当前标签页关闭，你对当前文件夹显示所作的修改都作废，通通恢复到默认的设置。

DOpus的逻辑是这样的：如果你没有特定指明哪个文件夹需要特殊对待，那么我就用默认的形式来给你显示，这样的话我需要记的东西比较少，占用的系统资源也不高。

那么，我们如何来保存对文件显示栏的修改呢？这就需要使用4.20中提到的文件夹选项了。我用一个例子来说明如何操作。如下左图，我增加了「文件总数」一栏，并且按照最后修改时间反序排列。但是如果我新打开一个标签页，还是同样的路径，显示的样式又回到了默认，只剩名称、大小、类型、修改时间及属性五列，按名称正序，如下右图：
